# Random Quran Verse 🌙

A simple and responsive web app that displays a **random Quranic verse (ayah)** every time you press **Start**. Perfect for daily inspiration, reflection, or learning.

## 💡 Features
- Display a random Quranic verse (ayah) on **Start**.  
- Multiple themes to choose from.  
- Fully **responsive** for mobile, tablet, and desktop.  
- Clean and minimal design for easy reading.  

---

## 🛠️ Tech Stack
- **HTML**  
- **CSS**  
- **Bootstrap**  

---

## ⚡ How to Use
1. Open the app in your browser.  
2. Press the **Start** button.  
3. Enjoy a new random verse every time.
